# Web (Next.js Arabic RTL)

واجهة عربية RTL حديثة تحتوي على:

- Dashboard
- Tasks
- Projects
- Employees (HR)
- Employees (HR Full Management + Import + Uploads)
- Goals
- Notifications
- Leaderboard
- Reports
- Attendance admin overview (for analytics roles)
- Audit Log
- Org Chart (hierarchical tree)
- Login with optional OTP verification
- Executive reports section

## Run

```bash
npm run dev
```

## API URL

يتم استخدام `NEXT_PUBLIC_API_URL`، والافتراضي:

- `http://100.10.10.10:4000/api`

## Attendance Debug

- `NEXT_PUBLIC_ATTENDANCE_DEBUG=true` لتفعيل سجلات تتبع الحضور/الموقع في Console للمتصفح.
